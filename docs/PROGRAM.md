# Program

본 프로젝트는 데이터베이스를 형성하는 것 이외에도, 네트워크 그래프 형태로 프로그램을 구성하고 이를 바탕으로 한국어 학습자를 돕는 데에 그 취지가 있습니다. 따라서 본 문서에서는 이를 위해 프로그램을 어떻게 구성할 지 논의한 결과 중 확정된 것을 나열합니다.

## 1. 프로그램 목적

본 프로젝트에서 만들어지는 프로그램은 한자 기반 한국어 학습을 돕는 학습 프로그램으로써, 각 한자를 사전처럼 넘기며 그 소리, 예문 등을 차례로 보고 익힐 수 있도록 구성되어야 합니다. 또한 각 한자는 그래프 이론을 바탕으로 내부적으로 네트워크 그래프 형태로 구성되어야 하며, 가능하다면 이 역시 시각화하여 언어를 시각적으로 학습할 수 있는 프로그램으로 발전시키는 것을 목표로 합니다.

## 2. 사용 언어

사용 언어에 제한을 두지는 않지만, 다음과 같은 공통의 목표를 지향하도록 합니다.

1. 우리의 프로그램은 속도가 빨라야 합니다. 1800개에 달하는 한자와 그 조합으로 파생되는 네트워크 그래프의 시각화는 생각보다 프로그램을 버겁게 만들 수 있습니다. 최대한 빠르고 병렬 처리 가능한 프로그램 언어와 프레임워크를 우선해야 합니다.
2. 우리의 프로그램은 접근이 쉬워야 합니다. 크로스 플랫폼 등으로 개발하거나, 혹은 웹 기반 접근을 가능하게 하여 언제 어디서나 학습이 가능하도록 지원해야 합니다.
3. 우리의 프로그램은 다중 언어를 지원해야 합니다. 우선은 한국어/영어 UI 지원을 최우선 목표로 하며, 이후 러시아어, 중국어 및 일본어 등으로 확장할 예정입니다.
